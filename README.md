# QuovTools Monorepo

AI-powered micro-SaaS portfolio under quovtools.com.

## Products
- **EmailForge** (email.quovtools.com) - AI Email Sequence Builder
- **PrepSync** (meet.quovtools.com) - AI Meeting Prep Summarizer
- **LegalGuard** (legal.quovtools.com) - AI Contract Clause Analyzer
- **TechResume** (resume.quovtools.com) - AI Resume Optimizer for Tech
- **CaptionAI** (social.quovtools.com) - AI Social Media Caption Generator

## Quick Start
```bash
npm install
npm run dev
```

## Scripts
- `npm run dev` - Run all apps in development
- `npm run build` - Build all apps
- `npm run lint` - Lint all packages
- `npm run type-check` - TypeScript check all packages
